DEM Data Visualization

Using C++ and OpenGL, calculates linear and quadratic splines given a Digital Elevation Model (DEM) and generates a wireframe model of the terrain. All DEM data is displayed in ESRI ASCII Grid format. The code can be viewed here, but requires openGL libraries in order to run correctly.

Lexomics Research Group

The Lexomics Research Group is comprised of Computer Science and English students and professors with a goal of analyzing digitized texts and works of literature. The main text-mining tool that we created is Lexos: An Integrated Lexomics Workflow, that is built using Flask, a python micro-framework for web application development. This tool includes various options of text manipulation and analysis techniques based on word frequencies and patters.

MedPING Simulator

Using the Pebble Smartwatch, this application simulates a hypothetical, futuristic scenario as if some sort of internal device streams vital sign data about a person's body. This application would receive this data, and alert the user if vital signs, such as blood pressure, glucose level, and body temperature, reached an unstable level. This code is written in C, using the Pebble Watch SDK.

Geometric Manifolds

Assuming that globally, the universe is some geometric manifold, its shape and structure could potentially be distinguished using Cosmic Microwave Background. Using the Beamer LaTeX class, I created a high-level slideshow presentation introducing some of these topics, accompanied by a longer paper explaining the ideas in more detail. The LaTeX code and PDF files of the paper and presentation can be found here.
